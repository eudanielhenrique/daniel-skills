# daniel-skills (v1.0)

Duas skills operacionais de backup e restore do Firestore para o Claude Code.

## Skills disponíveis

- `/dh-firestore-backup` — Exporta collections do Firestore para um arquivo JSON local
- `/dh-firestore-restore` — Restaura dados no Firestore a partir de um arquivo JSON de backup

Cada skill tem seu próprio SKILL.md na subpasta.
